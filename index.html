🔧 連接 Firebase 資料庫

此工具使用 Firebase Realtime Database 進行即時同步協作。
請按以下步驟設定：

① 建立 Firebase 專案
前往 console.firebase.google.com
新建專案 → 建立 Realtime Database（選 test mode）

② 設定資料庫規則（Rules 頁籤）：

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

③ 取得設定碼
專案設定 → 您的應用程式 → SDK → 複製 firebaseConfig 的 JSON 內容
